# Glow by Gaby — Catálogo web premium

Proyecto estático listo para publicar en Netlify.

## Mejoras incluidas

- Nuevo WhatsApp: **937 549 111**.
- Diseño premium en rosa, plata y blanco.
- Selector de idioma **Español / English**.
- Catálogo interactivo con filtros.
- Al agregar un producto, la tarjeta muestra inmediatamente:
  - cantidad actual;
  - botón para disminuir;
  - botón para aumentar;
  - estado visual de producto seleccionado.
- Carrito con cantidades editables.
- Formulario de pedido con:
  - nombre completo;
  - número de contacto;
  - ciudad o distrito;
  - dirección;
  - referencia;
  - modalidad de entrega;
  - horario;
  - método de pago;
  - notas;
  - autorización de contacto.
- Mensaje formal y detallado para WhatsApp con productos, cantidades, entrega, pago y confirmaciones solicitadas.
- Galería sin recortes y con ampliación de fotografías.
- Favicon e iconos propios con la letra **G**.
- Adaptación para celulares pequeños, iPhone, Android, tablet y computadora.
- Enlace de apoyo a **Mi Pequeño Espacio**:
  https://jairo-web-dev.netlify.app/

## Publicar en Netlify

1. Descomprime `Glow-by-Gaby-Web-Premium.zip`.
2. Entra a Netlify.
3. Arrastra la carpeta `glow-by-gaby-web` al área de despliegue manual.
4. Espera a que Netlify genere el enlace.

## Archivos principales

- `index.html`: estructura de la página.
- `styles.css`: diseño responsive y apariencia premium.
- `script.js`: carrito, cantidades, idiomas, galería y WhatsApp.
- `products.js`: información en español e inglés de los productos.
- `favicon.svg`, `favicon.ico`, `apple-touch-icon.png`: iconos de la G.
- `site.webmanifest`: configuración para móviles.
